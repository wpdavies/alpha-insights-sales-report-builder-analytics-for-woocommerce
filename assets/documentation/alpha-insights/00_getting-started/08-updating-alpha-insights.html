Updating Alpha Insights

Keep Alpha Insights up to date to get the latest features and improvements. This quick guide shows you how to update the plugin safely.

Checking for Updates

  1. WordPress shows notification when update available
  2. Go to Plugins → Installed Plugins
  3. Look for "Update Available" under Alpha Insights

How to Update

  1. Click Update Now
  2. Wait 30-60 seconds for completion
  3. See "Updated successfully" message

After Updating

  • Clear browser cache (Ctrl+Shift+Delete)
  • Check dashboard loads correctly
  • Open a report to verify data displays

Best Practice

Backup your site before major updates (use backup plugin or hosting backup tool)

Auto-Updates

Enable automatic updates on Plugins page for hands-free updating
