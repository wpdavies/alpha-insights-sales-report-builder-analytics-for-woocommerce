Managing Your License

Your Alpha Insights license gives you access to updates, support, and premium features. This guide covers checking license status, moving licenses between sites, and managing renewals.

Viewing License Status

  1. Go to Alpha Insights → Settings → License
  2. See license information panel

License Information Displayed

  • License Status: Active, Expired, or Invalid
  • License Type: Your plan (Pro Annual, Pro Lifetime, Agency, etc.)
  • Registered To: Email address
  • Expires: Renewal date (or "Lifetime" for lifetime licenses)
  • Activations Used: E.g., "1 of 1 sites" or "3 of 5 sites"
  • Last Checked: When status was last verified

Managing Site Activations

Checking Activations

See how many sites are using your license:

  • Single Site: "1 of 1 sites"
  • 5 Site: "3 of 5 sites" (2 available)
  • Unlimited: "12 of Unlimited sites"

Activation Limit Reached

Error:"License activation limit reached"

Cause: All activation slots used

Solutions:

  1. Deactivate unused sites: Free up activation slots
  2. Upgrade license: Get more activation slots
  3. Contact support: Request reset if you lost access to old sites

Moving License to New Site

Standard Process

Step 1: Deactivate on Old Site

  1. Log into old site's WordPress
  2. Go to Alpha Insights → Settings → License
  3. Click Deactivate License
  4. Wait for confirmation

Step 2: Activate on New Site

  1. Install Alpha Insights on new site
  2. Go to Settings → License
  3. Enter your license key
  4. Click Activate License

Processing time: Immediate (30 seconds max)

Lost Access to Old Site

If you can't access old site to deactivate:

  1. Go to your WP Davies account
  2. Navigate to Licenses section
  3. Find Alpha Insights license
  4. Click Manage Activations
  5. Deactivate old site remotely

Alternative: Contact support to manually reset activations

Upgrading Your License

From Single Site to Multi-Site

  1. Go to WP Davies Account → Subscriptions
  2. Click Upgrade next to your subscription
  3. Choose new plan (5 Site or Unlimited)
  4. Pay difference (prorated)
  5. License updates immediately

Multiple Licenses

Managing Multiple Purchases

If you have multiple Alpha Insights licenses:

  • Each license has unique key
  • Use different license on each site (or set of sites)
  • All managed from your WP Davies account

Viewing All Licenses

  1. WP Davies Account → Licenses
  2. See all your licenses
  3. Each shows: Status, Expires, Activations Used

Troubleshooting

Issue: "License invalid" after renewal

Solution:

  1. Go to Settings → License
  2. Click Check License
  3. Status updates to Active

Issue: Can't deactivate on old site

Solution: Deactivate remotely from WP Davies account

Issue: Lost license key

Solution:

  • Check purchase confirmation email
  • Log into WP Davies account → Licenses
  • License key displayed there

Issue: License shows expired but I renewed

Solutions:

  • Click Check License to refresh
  • Verify renewal payment processed
  • Check WP Davies account shows active
  • Contact support if still showing expired

License Types Comparison

Single Site License

  • 1 activation
  • Perfect for single store owners
  • All features included
  • Annual renewal

5 Site License

  • 5 activations
  • Good for multi-store owners or small agencies
  • Same price per site as 2.5 single licenses

Unlimited Sites License

  • Unlimited activations
  • Perfect for agencies
  • Use on all client sites
  • Best value for 10+ sites

Lifetime License

  • One-time payment
  • Never expires
  • Lifetime updates and support
  • Available for Single or Unlimited sites

Best Practices

  • Keep license activated on live sites only (not staging/dev)
  • Deactivate before deleting plugin or moving sites
  • Set calendar reminder for renewal date
  • Keep license key in password manager
  • Document which sites use which licenses (if managing many)
