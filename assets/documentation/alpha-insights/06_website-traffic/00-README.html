💡 Quick Start

Want to enable website tracking in 5 minutes?

  1. Read: Understanding Website Analytics (section: Enabling Website Analytics)
  2. Go to: Alpha Insights → Settings → General Settings
  3. Enable: "Enable WooCommerce Event Tracking" = True
  4. Exclude: Your admin role from tracking
  5. Save: Click "Save Changes"
  6. Test: Visit your site in incognito mode, check Realtime Dashboard
  7. Next: Set up UTM tracking for campaigns
