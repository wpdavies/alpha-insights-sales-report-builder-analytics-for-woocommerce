Understanding Pre-Built Reports

Alpha Insights includes 20+ pre-built reports covering all aspects of your business - from sales and products to customers, marketing, and website traffic. These reports are ready to use immediately and provide instant insights without any configuration.

Why Use Pre-Built Reports?

  • Instant insights: No setup required, start analyzing immediately
  • Best practices: Built by eCommerce experts using proven report structures
  • Comprehensive coverage: Every aspect of your business covered
  • Learning tool: Study them to understand how to build your own custom reports
  • Time-saving: Common questions already answered

Accessing Pre-Built Reports

  1. Go to Alpha Insights → Reports
  2. You'll see all available reports organized by category
  3. Click any report to open it

Report Categories

Reports are grouped into logical categories:

  • Sales Reports: Revenue, orders, and sales performance
  • Product Reports: Product performance and inventory insights
  • Customer Reports: Customer behavior and lifetime value
  • Marketing Reports: Campaign performance and advertising ROI
  • Website Traffic: Visitor analytics and traffic sources
  • Financial Reports: Profit & loss, expense analysis

Using Pre-Built Reports

Customize Date Range

All pre-built reports allow date range changes:

  1. Open report
  2. Click date range selector at top
  3. Choose different period
  4. Report updates automatically

Apply Additional Filters

Filter pre-built reports:

  1. Click Filters button
  2. Add filters (products, categories, traffic sources, etc.)
  3. Apply filters
  4. Report shows filtered data

Modify Widgets

Enter edit mode to customize:

  1. Click Edit Report
  2. Add, remove, or modify widgets
  3. Rearrange layout
  4. Save as custom copy (preserves original)

Export and Share

  • Export pre-built reports as PDF, CSV, or Excel
  • Create share links for team access
  • Schedule automated email delivery

Best Practices

Start with Pre-Built

  • Explore all pre-built reports first
  • Understand what each shows
  • Identify which answer your questions
  • Only create custom reports for unique needs

Daily/Weekly Review Routine

Daily (5 minutes):

  1. Store Dashboard - Quick health check
  2. Recent Orders - Any issues?

Weekly (30 minutes):

  1. Sales Overview - Trend analysis
  2. Products - Profitability review
  3. Campaigns - Ad performance check

Monthly (1-2 hours):

  1. Profit & Loss Statement - Financial review
  2. Customers - Segment analysis
  3. Acquisition Sources - Channel optimization

Combine Multiple Reports

  • Use Store Dashboard for overview
  • Drill into Products report for details
  • Check Campaigns for marketing context
  • Review Customers for audience insights

Customizing Pre-Built Reports

Save as Custom Copy

To modify without losing original:

  1. Open pre-built report
  2. Click Duplicate
  3. Report copies to your custom reports
  4. Edit the copy freely
  5. Original remains unchanged

Create Variations

Examples:

  • Store Dashboard → Create "Store Dashboard - Yesterday Only"
  • Products → Create "Products - Clothing Category Only"
  • Campaigns → Create "Campaigns - December Holiday Period"

Report Update Cycle

How Often Reports Update

  • Realtime Dashboard: Every 30 seconds (when viewing)
  • Other reports: When you refresh or reload
  • Cached data: May be 5-15 minutes old (click Refresh for latest)

Data Refresh

  1. Click Refresh button at top of any report
  2. Data reloads from database
  3. See most current information
