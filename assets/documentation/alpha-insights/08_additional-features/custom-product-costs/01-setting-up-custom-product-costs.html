Setting Up Custom Product Costs

Beyond standard product costs, Alpha Insights supports advanced cost calculation methods for complex scenarios like variable costs, tiered pricing, or dynamic cost calculations.

Standard Product Cost

Simple cost per unit:

  1. Edit product in WooCommerce
  2. Find Alpha Insights tab
  3. Enter cost (e.g., $25.00)
  4. Save

Variable Product Costs

Different costs for different variations:

  1. Edit variable product
  2. Go to Variations tab
  3. Expand each variation
  4. Enter cost specific to that variation

Example:

  • Small T-Shirt: $10.00 cost
  • Large T-Shirt: $12.00 cost
  • XL T-Shirt: $14.00 cost

Tiered Costs Based on Quantity

Cost changes based on order quantity:

  1. Enable Quantity-Based Costs in product settings
    • 1-10 units: $15.00 each
    • 11-50 units: $12.00 each
    • 51+ units: $10.00 each

Time-Based Costs

Costs that change over time (seasonal, supplier changes):

  1. Set cost with effective date
  2. Historical orders use cost from that date
  3. Future orders use current cost

Advanced Cost Options

Include in Cost Calculation

Choose what to include:

  • Base product cost
  • Addon costs (product add-ons)
  • Custom fees

Cost Allocation Methods

  • Per unit: Standard method
  • Per order: Split evenly across line items
  • Weighted by price: Higher-priced items get more cost allocated
