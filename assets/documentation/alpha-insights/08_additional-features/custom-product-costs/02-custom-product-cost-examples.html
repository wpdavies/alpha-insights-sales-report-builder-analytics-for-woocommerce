Custom Product Cost Examples

Example 1: Wholesale Quantity Discounts

Scenario: Bulk buyers get lower costs

Setup: 

	1-5 units: $25 cost each 6-20 units: $22 cost each 21+ units: $19 cost each

Example 2: Seasonal Cost Changes

Scenario: Supplier raises prices in summer

Setup:

  • Jan-May: $20.00 cost
  • Jun-Aug: $23.00 cost (summer premium)
  • Sep-Dec: $20.00 cost

Example 3: Size-Based Costs

Scenario: Apparel costs vary by size

Variable Product Setup:

  • Size S: $12 cost
  • Size M: $13 cost
  • Size L: $14 cost
  • Size XL: $15 cost
  • Size XXL: $17 cost

Example 4: Color-Based Costs

Scenario: Special colors cost more to produce

  • Standard colors (black, white): $18 cost
  • Premium colors (red, blue): $20 cost
  • Custom colors: $25 cost

Example 5: Bundled Product Costs

Scenario: Product bundles have combined costs

Bundle: T-Shirt + Hat

  • T-Shirt cost: $15
