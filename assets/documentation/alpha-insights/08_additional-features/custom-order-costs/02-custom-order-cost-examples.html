Custom Order Cost Examples

Example 1: Gift Wrapping Service

Scenario: You offer gift wrapping for $7.50

Setup:

  • Cost Type: "Gift Wrapping"
  • Amount: $7.50
  • Apply: When customer selects gift wrap option

Example 2: Insurance

Scenario: Add insurance for orders over $200

Setup:

  • Cost Type: "Shipping Insurance"
  • Amount: 2.5% of order total
  • Rule: Auto-apply if order >$200

Example 3: Rush Processing

Scenario:$15 fee for expedited orders

Setup:

  • Cost Type: "Rush Processing"
  • Amount: $15.00
  • Apply: Manual when customer requests rush

Example 4: International Customs Handling

Scenario:$12 handling fee for international orders

Setup:

  • Cost Type: "Customs Processing"
  • Amount: $12.00
  • Rule: Auto-apply if shipping country US

Example 5: Custom Assembly

Scenario: Labor cost for complex product assembly

Setup:

  • Cost Type: "Custom Assembly"
  • Amount: $25.00
  • Apply: Manual for specific products
