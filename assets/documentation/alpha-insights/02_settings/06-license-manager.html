License Manager

Manage your Alpha Insights license activation, view license details, and handle license transfers. Your license key enables updates, support, and ensures your plugin remains functional.

Accessing License Settings

  1. Go to Alpha Insights → Settings
  2. Navigate to the License settings page
  3. You'll see your license management interface

Why License Management Matters

  • Plugin updates: Access to latest features and bug fixes
  • Priority support: Get help when you need it
  • Continued functionality: Active license required for plugin to work
  • Multiple sites: Manage license usage across sites
  • Renewal management: Track expiration and renewals

License Status Overview

Your License Status Bar

At the top of the page, you'll see your current license status:

Possible statuses:

  • Active: License is valid and working (green) ✅
  • Inactive: License entered but not activated (yellow) ⚠️
  • Expired: License subscription has expired (red) ❌
  • Invalid: License key not recognized (red) ❌

Status bar also shows:

  • Current license status in bold
  • Quick action button (Activate, Deactivate, or Renew)
  • Refresh License button

Your License Key

Entering Your License Key

Where to find your license key:

  • In your purchase confirmation email
  • On your account page at WP Davies
  • In your account → Downloads or Subscriptions section

To enter license key:

  1. Copy your license key from email or account page
  2. Paste into "Your License Key" field
  3. Click Save Changes
  4. Alpha Insights will attempt automatic activation
  5. If not auto-activated, click Activate License button

License key format: Usually looks like abc123-def456-ghi789-jkl012

License Validation

When you save a license key, Alpha Insights:

  1. Validates format is correct
  2. Checks key with WP Davies license server
  3. Verifies key is valid and not revoked
  4. Attempts activation if license is valid
  5. Shows success or error message

License Details

License Information Display

License Status

  • Shows current status: active, inactive, expired, etc.
  • Updates after activation or renewal

License Owner

  • Name of person who purchased license
  • Shows first and last name
  • Helps verify license is legitimate

Max Number Of Uses

  • How many sites this license allows
  • Common: 1, 3, 5, or unlimited
  • Depends on license plan purchased

Number Of Uses Remaining

  • How many activations are still available
  • Decreases when you activate on a new site
  • Increases when you deactivate from a site
  • If 0, you need to deactivate from another site first

Example:

  • Max: 3 sites
  • Currently activated on: 2 sites
  • Remaining: 1 more site available

Expiration Date

What it shows: When your license subscription expires

How it works:

  • Active subscription: Date is in the future
  • Each successful subscription payment extends expiration date
  • Automatically updates when subscription renews
  • If expired, shows past date in red

Display format: "Friday 15th March, 2025"

Important: You'll need to renew before this date to maintain access

Last Updated

What it shows: When license data was last checked with license server

Format: "Friday 22nd October, 2024 at 3:45pm"

Why it matters:

  • Helps verify license data is current
  • Recent timestamp means data is up-to-date
  • Old timestamp might indicate sync issues

License Actions

Activate License

When to use: After entering a license key that shows "inactive" status

Process:

  1. Enter license key in field
  2. Click Save Changes
  3. If not auto-activated, click Activate License button
  4. Alpha Insights contacts license server
  5. License is activated for this site
  6. Status updates to "Active"
  7. Plugin is fully functional

Success message: "License key activated successfully"

Deactivate License

When to use:

  • Moving Alpha Insights to a different site
  • No longer using plugin on this site
  • Need to free up an activation slot
  • Troubleshooting license issues

Process:

  1. Click Deactivate License button
  2. Confirm deactivation
  3. License is deactivated for this site
  4. Activation slot becomes available
  5. Status changes to "Inactive"
  6. Plugin may enter limited functionality mode

Important: Deactivate before moving site or deleting plugin to free up activation

Refresh License

What it does: Immediately checks license status with server

When to use:

  • Just renewed subscription
  • License details seem outdated
  • Troubleshooting license issues
  • After making changes on WP Davies account

Process:

  1. Click Refresh License button
  2. Alpha Insights contacts license server
  3. Latest license data is retrieved
  4. Display updates with current information
  5. "Last Updated" timestamp refreshes

Use case example: You just renewed your subscription - click Refresh to update expiration date

License Expiration

Expired License Notice

If your license has expired, you'll see a prominent notice at the top:

Notice includes:

  • Title: "Your License Has Expired"
  • Message: Explanation that renewal is needed
  • Renewal discount: "Use code 'RENEWAL20' to take 20% off your next payment"
  • Action button: "Click here to renew" linking to renewal page

What Happens When License Expires

  • Plugin continues working: Core functionality remains
  • No updates: Can't receive plugin updates
  • No support: Support access is suspended
  • Security risk: Missing security updates
  • Feature updates: New features not available

Recommendation: Renew before expiration to maintain uninterrupted service

Renewing Your License

  1. Click Renew License button (appears when expired)
  2. Or visit WP Davies My Account → Subscriptions
  3. Find Alpha Insights subscription
  4. Click "Renew" or "Reactivate"
  5. Use code "RENEWAL20" for 20% discount
  6. Complete payment
  7. Return to Alpha Insights License Manager
  8. Click Refresh License
  9. Expiration date and status update automatically

Managing Your License

Access Your WP Davies Account

To manage license online:

  1. Click Manage Your License button on settings page
  2. Or visit wpdavies.dev/my-account/
  3. Log in with your account credentials
  4. Access subscriptions, downloads, and license management

What you can do in your account:

  • View all active licenses
  • See which sites each license is activated on
  • Deactivate licenses remotely
  • Update payment information
  • Change subscription plan
  • Download plugin versions
  • Access documentation and support

Transferring License to New Site

Scenario: You want to move Alpha Insights from Site A to Site B

Process:

  1. On Site A (old site):
    • Go to License Manager
    • Click Deactivate License
    • Confirm deactivation
    • One activation slot now available
  2. On Site B (new site):
    • Install Alpha Insights plugin
    • Go to License Manager
    • Enter your license key
    • Click Save Changes
    • Click Activate License
    • License now active on new site

Important: Always deactivate before moving to avoid using up activations

Multiple Site Management

If you have a multi-site license (3, 5, or unlimited sites):

Track activations:

  • Check "Number Of Uses Remaining" on any site
  • Or log into WP Davies account to see all active sites

Best practices:

  • Document which sites use the license
  • Deactivate from staging/dev sites when not in use
  • Keep activation slots available for flexibility
  • Consider upgrading if you run out of activations

Troubleshooting

License Key Not Working

Check:

  • License key copied correctly (no extra spaces)
  • License purchased from WP Davies (not third-party)
  • License is for Alpha Insights (not different product)
  • Subscription hasn't been canceled
  • Payment for subscription is up to date

Solution: Visit WP Davies account to verify license is valid

"Maximum Number of Activations Reached"

Cause: License is already active on maximum allowed sites

Solution:

  1. Log into WP Davies account
  2. View active sites for this license
  3. Deactivate license from unused sites
  4. Or deactivate from old/staging sites
  5. Return to new site and activate

Alternative: Upgrade to license plan with more activations

License Shows as Expired But Was Just Renewed

Solution:

  1. Click Refresh License button
  2. Wait for data to update from server
  3. If still showing expired, check WP Davies account
  4. Verify subscription payment processed successfully
  5. Allow up to 24 hours for payment processing

"Could Not Connect to License Server"

Causes:

  • Internet connection issue
  • Firewall blocking outbound connections
  • Server temporarily unavailable
  • SSL certificate issues

Solutions:

  • Check your site has internet connectivity
  • Wait a few minutes and try again
  • Contact hosting provider about firewall rules
  • Verify server can make outbound HTTPS requests

License Deactivated Automatically

Possible causes:

  • Payment failed for subscription
  • Subscription was canceled
  • Refund was issued
  • License was revoked

Solution:

  1. Check WP Davies account for subscription status
  2. Verify payment method is valid
  3. Update payment info if needed
  4. Reactivate subscription if canceled
  5. Contact support if issue persists

License Plans

Understanding License Tiers

Alpha Insights typically offers several license plans:

Single Site License:

  • 1 site activation
  • All features included
  • Updates and support
  • Perfect for single store

Multi-Site Licenses (3, 5, 10+ sites):

  • Multiple activations
  • Same features as single site
  • Better value per site
  • Great for agencies or multiple stores

Unlimited License:

  • Unlimited activations
  • All features
  • Best for agencies
  • Use on all client sites

Upgrading Your License

To upgrade from single to multi-site license:

  1. Visit WP Davies account
  2. Go to Subscriptions
  3. Find Alpha Insights subscription
  4. Look for "Upgrade" option
  5. Choose new plan tier
  6. Pay difference
  7. Activations increase immediately

Best Practices

License Management

  • Activate immediately: Don't delay after purchase
  • Document license key: Store securely for future reference
  • Track activations: Know which sites use your license
  • Deactivate properly: Always deactivate before moving sites
  • Monitor expiration: Set calendar reminder to renew

Multiple Sites

  • Use staging-specific license for dev sites (if available)
  • Deactivate from staging when not actively developing
  • Keep production sites activated permanently
  • Document which site uses which activation

Renewal Management

  • Enable auto-renewal in WP Davies account
  • Keep payment method up to date
  • Check email for renewal reminders
  • Use RENEWAL20 code for 20% discount
  • Renew before expiration to avoid service interruption

Support and Contact

Getting Help with License Issues

If you can't resolve license issues:

  1. Check WP Davies documentation
  2. Review this guide thoroughly
  3. Try refreshing license
  4. Check WP Davies account status
  5. Contact WP Davies support with:
    • License key
    • Site URL
    • Description of issue
    • Screenshots of error messages

Support channels:

  • WP Davies website contact form
  • Support ticket system
  • Email support (check your purchase email)
