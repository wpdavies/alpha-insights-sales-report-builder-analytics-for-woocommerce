Integrations

The Integrations page is where you view and configure all Alpha Insights integrations in one place. From here you can connect advertising platforms, shipping providers, webhooks, and other services that feed data into your profit reports.

Accessing Integrations

  1. Go to Alpha Insights → Settings in your WordPress dashboard
  2. Click Integrations in the settings submenu
  3. The URL is Settings → Integrations (?page=...&subpage=integrations)

What You See

When you open Integrations, you see a single page with:

  • Header: “Integrations” at the top
  • Sidebar (left): A search box and category tabs (e.g. All, Marketing, Data Management, Shipping). Use search to filter by name; use tabs to filter by category.
  • Grid (main area): A card for each available integration. Each card shows the integration name, short description, and (for Pro integrations) a Pro badge. Integrations you have access to can be clicked to open their settings.

When you click an integration card, the grid is replaced by that integration’s settings form. The header then shows Return To Integrations and Save Changes. Use “Return To Integrations” to go back to the grid without saving; use “Save Changes” to save and stay on that integration, or save and then return.

Available Integrations

Which integrations appear depends on your plan and setup. Typical options include:

  • Facebook Ads (Marketing) – Campaign conversion reporting and ad spend tracking
  • Google Ads (Marketing) – Campaign conversion reporting and ad spend tracking
  • Webhooks (Data Management) – Send data to an external URL (e.g. Zapier, Make, Slack) on a schedule
  • ShipStation (Shipping) – Sync shipping costs with orders via the ShipStation API
  • StarShipIt (Shipping) – Sync shipping costs with orders via the StarShipIt API

Pro-only integrations show a Pro badge; their settings are available when you have Alpha Insights Pro and the integration is enabled.

Using the Page

  • Open an integration: Click its card. The URL will include ?integrations=slug (e.g. integrations=webhooks).
  • Save settings: Change any options, then click Save Changes. Some integrations may show a success message or refresh the form.
  • Go back to the list: Click Return To Integrations to see all cards again.
  • Search: Type in the sidebar search box to show only integrations whose name or description matches.
  • Filter by category: Click a category tab (e.g. Marketing, Shipping) to show only integrations in that category.

Webhooks and Other Integrations

Webhook configuration (URL, schedule, logging, test) is no longer on a separate “Webhook Settings” page. It lives under Settings → Integrations: click the Webhooks card to open the webhook settings. The same pattern applies to other integrations: each has its own form reached from the Integrations grid. For detailed webhook setup, see Webhook setup in the Integrations documentation.
